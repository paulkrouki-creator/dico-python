Les types de données : En Python, les types de données permettent de représenter et de manipuler différentes catégories d’informations. Parmi les types de base, on trouve les entiers (int), utilisés pour les nombres sans partie décimale ; les nombres décimaux (float) pour les valeurs décimales ; les chaînes de caractères (str) pour le texte ; et les booléens (bool) qui prennent les valeurs True ou False. Python propose également des structures de données plus complexes, comme les listes (list), qui stockent des collections ordonnées et modifiables d’éléments ; les tuples (tuple), similaires aux listes mais immuables ; les ensembles (set), qui contiennent des éléments uniques ; et les dictionnaires (dict), qui associent des clés à des valeurs. Grâce à cette diversité de types de données, Python permet de développer des programmes flexibles et adaptés à de nombreux besoins, tout en facilitant la gestion et l’organisation des informations. On peut tester le type d'un objet avec la fonction type(). int (nombres entiers) : Le type int est utilisé pour représenter les nombres entiers, c’est-à-dire les nombres sans partie décimale, comme -5, 0 ou 42. Il est couramment utilisé pour les calculs, les compteurs et les indices dans les structures de données. Les variables de type int peuvent être créées directement en attribuant une valeur entière ou en convertissant une autre donnée à l’aide de la fonction int(). Voir plus écriture scientifique float (nombres décimaux) : Le type float permet de représenter les nombres à virgule flottante, c’est-à-dire les nombres comportant une partie décimale, comme 3.14, -0.5 ou 2.0. Il est utilisé pour les calculs nécessitant une précision décimale. Les variables de type float peuvent être créées directement en attribuant une valeur décimale ou en convertissant une autre donnée à l’aide de la fonction float(). complex (nombres complexes) : Le type complex permet de représenter des nombres complexes composés d’une partie réelle et d’une partie imaginaire, comme (3 + 2j). Il est principalement utilisé dans les calculs scientifiques et mathématiques. Les variables de type complex peuvent être créées directement en utilisant la notation complexe ou en convertissant des valeurs à l’aide de la fonction complex(). str (chaines de caractère) : Le type str (string) est utilisé pour stocker du texte sous forme de chaînes de caractères. Les chaînes peuvent contenir des lettres, des chiffres ou des symboles, comme "Bonjour" ou "Python". Les variables de type str peuvent être créées directement en utilisant des guillemets (" et " ou ' et ' ou ''' et ''' ou """ et """) ou en convertissant une autre donnée à l’aide de la fonction str(). Voir plus f-sting list : Le type list permet de stocker une collection ordonnée d’éléments. Les listes sont modifiables et peuvent contenir des données de types différents. Les variables de type list peuvent être créées directement à l’aide de crochets [] ou en convertissant une séquence grâce à la fonction list(). tuple : Le type tuple est une collection ordonnée d’éléments similaire à une liste, mais dont le contenu ne peut pas être modifié après sa création. Les variables de type tuple peuvent être créées directement à l’aide de parenthèses () ou en convertissant une séquence grâce à la fonction tuple(). range : Le type range représente une séquence de nombres générée automatiquement. Il est souvent utilisé dans les boucles pour répéter une action un certain nombre de fois. Les variables de type range peuvent être créées à l’aide de la fonction range(), en précisant une valeur de départ, d’arrêt et éventuellement un pas. bytes : Le type bytes permet de manipuler des données binaires sous forme d’une séquence immuable d’octets. Il est utilisé pour le traitement de fichiers ou de données réseau. Les variables de type bytes peuvent être créées directement avec le préfixe b ou à l’aide de la fonction bytes(). bytearray : Le type bytearray est similaire à bytes, mais il est modifiable. Il est utile lorsque les données binaires doivent être modifiées après leur création. Les variables de type bytearray peuvent être créées à l’aide de la fonction bytearray(). memoryview : Le type memoryview permet d’accéder directement aux données binaires d’un objet sans les copier en mémoire, ce qui améliore les performances. Les variables de type memoryview peuvent être créées à partir d’un objet binaire à l’aide de la fonction memoryview(). dict : Le type dict stocke des données sous forme de paires clé-valeur. Chaque clé est unique et permet d’accéder rapidement à la valeur associée. Les variables de type dict peuvent être créées directement avec des accolades {} ou à l’aide de la fonction dict(). bool : Le type bool représente une valeur logique pouvant être soit True, soit False. Il est utilisé dans les conditions et les comparaisons. Les variables de type bool peuvent être créées directement avec les valeurs True ou False, ou en convertissant une donnée grâce à la fonction bool(). set : Le type set permet de stocker une collection non ordonnée d’éléments uniques. Il est souvent utilisé pour supprimer les doublons ou effectuer des opérations ensemblistes. Les variables de type set peuvent être créées à l’aide d’accolades {} contenant des éléments uniques ou grâce à la fonction set(). frozenset : Le type frozenset est une version immuable du type set. Une fois créé, son contenu ne peut plus être modifié. Les variables de type frozenset peuvent être créées à partir d’un ensemble ou d’une séquence grâce à la fonction frozenset(). NoneType : Le type NoneType est le type de l’objet spécial None, qui représente l’absence de valeur ou une valeur nulle. Il est souvent utilisé pour indiquer qu’une variable n’a pas encore reçu de donnée significative ou qu’une fonction ne retourne aucun résultat particulier. Comme None est un singleton (il n’existe qu’une seule instance de cet objet), on vérifie généralement sa présence avec l’opérateur is. 🔎L’utilisation correcte de None permet de gérer proprement les cas où une valeur est absente ou indéfinie dans un programme.
